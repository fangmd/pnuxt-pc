# pnuxt-pc

>使用 Nuxt 实现的 PC 端 Vue SSR 项目模版


1. bootstrap 作为基础 CSS 框架

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
